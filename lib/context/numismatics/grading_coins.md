# Coin Grading Reference (Sheldon Scale)

## GRADE OVERVIEW

| Grade | Abbrev | Description | Value vs VF |
|-------|--------|-------------|-------------|
| 1-3 | PO-AG | Barely identifiable | 5-15% |
| 4-6 | G | Good, major features visible | 15-25% |
| 8-10 | VG | Very Good, main features clear | 25-40% |
| 12-15 | F | Fine, moderate wear | 40-60% |
| 20-35 | VF | Very Fine, light wear | 60-100% |
| 40-45 | EF/XF | Extremely Fine, trace wear | 100-200% |
| 50-58 | AU | About Uncirculated | 150-500% |
| 60-62 | MS | Mint State, heavy marks | 300-600% |
| 63-64 | MS | Choice, light marks | 500-1500% |
| 65-66 | MS | Gem | 1000-10000% |
| 67+ | MS | Superb Gem | 5000%+ |

## PROOF COINS

| Grade | Description |
|-------|-------------|
| PR-60 | Proof with handling marks |
| PR-63 | Choice Proof |
| PR-65 | Gem Proof |
| PR-67+ | Superb Gem Proof |
| PR-70 | Perfect Proof (rare) |

## PROBLEM COINS (REDUCED VALUE)

| Problem | Value Reduction |
|---------|-----------------|
| Cleaned | -50% to -90% |
| Polished | -60% to -90% |
| Damaged | -70% to -95% |
| Environmental | -50% to -80% |
| Whizzed | -80% to -95% |
| Tooled | -70% to -95% |
| Mounted | -30% to -60% |

## GRADING SERVICES

| Service | Tier | Notes |
|---------|------|-------|
| PCGS | Top | Most trusted for US coins |
| NGC | Top | Equal to PCGS |
| ANACS | Second | Respected, lower premium |
| ICG | Second | Lower premium |
| Others | Third | May not add value |

## GRADE VERIFICATION TIPS

- Look for wear on highest points first
- Check luster under angled light
- Bag marks vs circulation wear differ
- Original surfaces vs cleaned
- Eye appeal matters at high grades
