# Paper Money Grading Reference (PMG Scale)

## GRADE OVERVIEW

| Grade | Description | Folds | Characteristics |
|-------|-------------|-------|-----------------|
| 1-10 | Poor to VG | 7+ | Heavy wear, soiling |
| 12-15 | Fine | 5-6 | Moderate wear, still crisp |
| 20-35 | Very Fine | 3-4 | Light wear, attractive |
| 40-45 | Extremely Fine | 2 | Minor folds, original crispness |
| 50-58 | About Uncirculated | 0-1 | Single fold or handling |
| 60-62 | Uncirculated | 0 | No folds, may have marks |
| 63-64 | Choice Uncirculated | 0 | Light handling only |
| 65-66 | Gem Uncirculated | 0 | Near perfect |
| 67+ | Superb Gem | 0 | Exceptional quality |

## EPQ DESIGNATION

**EPQ = Exceptional Paper Quality**

- Original paper quality intact
- No chemical treatment
- No pressing or ironing
- Adds 10-30% premium
- Most desirable designation

## PMG DESIGNATIONS

| Designation | Meaning |
|-------------|---------|
| EPQ | Exceptional Paper Quality |
| Star | Exceptional eye appeal |
| Net | Problems noted, grade adjusted |

## CONDITION ISSUES

| Issue | Impact |
|-------|--------|
| Folds | Primary grade determinant |
| Tears | Significant reduction |
| Stains | Moderate reduction |
| Writing | Major reduction |
| Tape | Severe reduction |
| Missing pieces | Very severe |

## TIPS FOR ASSESSMENT

- Count folds carefully
- Check for hidden tears at folds
- Look for chemical brightening
- Original colors = premium
- Margin centering matters
- Serial number clarity
