# /deploy-status

Check the current Vercel deployment status for RealWorth.ai.

## Command

```bash
vercel ls --limit 5
```

Or check recent deployments:

```bash
vercel inspect <deployment-url>
```

## What to Report

1. Current production deployment status
2. Most recent deployment URL and state
3. Any failed deployments in the last 5
4. Build time and any errors
