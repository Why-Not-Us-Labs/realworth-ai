# /lint-fix

Run ESLint with auto-fix on the codebase.

## Command

```bash
npm run lint -- --fix
```

## What to Report

1. Number of files with issues
2. Number of errors vs warnings
3. Issues that couldn't be auto-fixed (require manual intervention)
4. Confirm what was fixed automatically

## Common Fixes

- Unused imports removed
- Missing semicolons added
- Consistent quotes applied
- Trailing whitespace removed
